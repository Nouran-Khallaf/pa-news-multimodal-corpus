# Validation register

| Deliverable | Initial state | Required final evidence |
|---|---|---|
| Collector | T2 on local fixtures | T3 end-to-end run plus manual article/image checks |
| Raw capture | P0 before execution | P1 timestamps/hashes; P2 manual source comparison sample |
| Derived text | T1 before execution | Schema/hash validation and manual extraction audit |
| Text analysis | A0 | Concordance review, comparator audit and sensitivity checks |
| Political framing | A0 | Independent coding, codebook revision and adjudication |
| Topic model | A0 | Multi-seed stability plus human topic review |
| Image models | A0 | Stratified error analysis for OCR/caption/category/object outputs |
| Multimodal relations | A0 | Human annotation of relation and image function |

Use P0–P2 for provenance, T0–T3 for technical validation and A0–A3 for analytical validation. Model execution alone never raises analytical validation above A0.
